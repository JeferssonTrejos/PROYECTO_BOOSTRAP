Este proyecto consiste en un sitio web multipágina diseñado para funcionar como un portafolio dinámico utilizando el framework **Bootstrap** como herramienta principal para ser completamente funcional y responsivo en distintos dispositivos (móvil, tablet y escritorio), además de la función de alternar el portafolio entre los diferentes integrantes del equipo.

---

### Secciones Principales
#### **Home**
Incluye una sección "Hero", una introducción personal y una presentación de servicios mediante "cards".

#### **Portafolio**
Una galería que muestra los proyectos desarrollados, utilizando **modals** para desplegar los detalles de cada uno.

#### **Sobre mí**
Presenta una **línea de tiempo (timeline)** con la formación académica y experiencia, además de **barras de progreso (progress bars)** para ilustrar las habilidades.

#### **Contacto**
Contiene un formulario con clases de Bootstrap.
